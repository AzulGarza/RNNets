# RNNets 

## Instalación
1. `git clone https://github.com/datomicomx/RNNets.git`
2. `cd RNNets`
3. `conda env update`
4. `source activate datomico-NLP`

### Nota:
- El env **datomico-NLP** es el que estoy usando para todo el desarrollo de datómico exclusivo de NLP.
